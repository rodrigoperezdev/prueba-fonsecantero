A los miembros de Fonsecantero.

Esta la forma en que yo trabajaría un proyecto ya existente que requiere una actualización utilizando las mismas tecnologías, manteniendo el código CSS y JS y de esta forma hacerlo crecer.

Para esta página en particular yo trabajaría en algunas implementaciones para buscar modernizar su entorno de trabajo, hacer mas eficiente el desarrollo y adquirir algunas nuevas funcionalidades que pueden incluso afectar el SEO.

Lo primero sería re-factorizar el código para cambiar nombres de clases y ids de css por aquellas que estén de acuerdo a la metodología BEM para hacer mas fácil el trabajo entre varios desarrolladores, actualizaría la version de Bootstrap a la mas reciente ya que esta es una version ya bastante antigua y tiene muchas propiedades de CSS que ya se des-actualizaron lo cual puede considerarse entre otras cosas una mala practica de SEO.

Por ultimo tener en cuenta todos los requerimientos de accesibilidad como textos alternativos en imágenes, niveles de contraste en el texto etc.

La parte de noticias con el buscador la realice haciendo un fetch a una API de noticias.

No se si tenía tiempo limite pero como se acerca el fin de semana quise hacerlo lo mas rápido posible para poderles entregar un resultado a la brevedad. De verdad estoy muy interesado en el puesto, soy muy buen autodidacta y me puedo adaptar a tiempos de trabajo de Fonsecantero. He posicionado 2 paginas en proyectos de mas de 8 meses. Logre llevarlos a los primeros puestos en Google con buenas practicas de SEO y estrategias de backlinks. Las paginas son instindog.com y tecfujal.mx

También se ReactJs y puedo moverme rápido a otro framework de front como vue.js. Adicionalmente estoy aprendiendo Nodejs para backend.

¡Gracias!
